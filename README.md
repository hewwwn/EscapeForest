## 🎮 Game Overview

**EscapeForest**는 3D 1인칭 생존 탈출 게임입니다.  
플레이어는 좀비가 가득한 공간에서 **5개의 열쇠를 수집하여 출구를 열고 탈출**해야 합니다.

탄약은 무한하지 않습니다.  
맵 곳곳에서 총알을 직접 찾아야 하며, 전략적인 플레이가 요구됩니다.

---

## 🔥 Features

- 🗝️ **5 Key Collection System**
  - 모든 열쇠를 수집해야 탈출 가능

- 🔫 **Gun & Ammo System**
  - 기본 무기: 총
  - 탄약은 맵에서 직접 획득

- 🧟 **Zombie AI Tracking**
  - 일정 거리 내 접근 시 플레이어 감지
  - 감지 후 지속적으로 추적

- 🌍 **3D Survival Environment**
  - 탐색 + 생존 + 탈출 요소 결합

---

## 🎯 Objective

1. 맵을 탐색한다.
2. 5개의 열쇠를 모두 찾는다.
3. 좀비를 피해 생존한다.
4. 출구를 열고 탈출한다.

---

## 🛠️ Tech Stack

- **Engine**: Unity
- **Language**: C#
- **Genre**: First-Person Survival Escape
- **Team Size**: 4 Members
